giddyup.tables documentation

Bootstrap table markup.

row

(row attr-map? & cells)
Returns a table row for `cells`.

### Example

    (row "Joe" "joe@example.com")

table

(table attr-map? caption-or-headers & rows)
Returns a table with optional caption, header row, and body rows.

### Example

    (table "Contact Info"
           ["Name" "Email"]
           ["Joe" "joe@example.com"]
           ["Jill" "jill@example.com"])

tbody

(tbody attr-map? & rows)
Returns a table body with `rows` passed through `row`.

### Example

    (tbody ["Joe" "joe@example.com"]
           ["Jill" "jill@example.com"])

thead

(thead attr-map? & headers)
Returns a table header section for `headers`.

### Example

    (thead "Name" "Email")
