giddyup.navigation documentation

Bootstrap navigation elements.

collapse-toggle

(collapse-toggle attr-map? target)(collapse-toggle attr-map? target alt-text)
Returns a collapse toggle button for use in navbars.

dropdown-menu

(dropdown-menu attr-map? & items)
Returns a dropdown menu consisting of `items`. For format of `items`, see
`dropdown-menu-item`.

### Item format

`items` can be one of:

* `:divider`
* `"Nav header"`
* `["link text" "/link-url"]`

### Example

    (dropdown-menu ["Add new" "/media/new"]
                   :divider
                   "View old media"
                   ["Photos" "/media/photos"]
                   ["Video" "/media/video"])

navbar

(navbar attr-map? & content)
Returns a navigation bar with `content`.

### Example

    (navbar
      [:div.navbar-header
       (link-to {:class "brand"} "/" "My Journal")
       (collapse-toggle "#main-nav")])

pager

(pager attr-map? prev-link next-link)
Returns a pager element. Disables the previous link and/or the next link if
passed strings for `prev-link` or `next-link`.

### Example

    (pager "Previous" (link-to "?page=2" "Next"))
