giddyup.forms documentation

Bootstrap form elements.

actions

(actions attr-map? & content)
Returns the `content` in a wrapper intended for form buttons.

### Example

    (actions
     (button-link-to "/" "Cancel")
     (submit-button "Sign up"))

button-link-to

(button-link-to attr-map? url & content)
Returns a button-like link for `url` with `content` as text.

### Example

    (button-link-to "/" "Cancel")

check-box

(check-box attr-map? name)(check-box attr-map? name checked?)
Returns a checkbox element.

### Example

    (check-box :remember-me)

drop-down

(drop-down attr-map? name options)(drop-down attr-map? name options selected)
Returns a dropdown element.

### Example

    (drop-down :language ["Finnish" "English"])

email-field

(email-field attr-map? name)(email-field attr-map? name value)
Returns an email element.

### Example

    (email-field :email)

error

(error attr-map? & content)
Returns an error element with `content` as text.

### Example

    (error "This field is required.")

fieldset

(fieldset attr-map? s & content)
Returns a fieldset wrapped around `content` with `s` as legend.

### Example

    (fieldset "Contact Info"
              (label :email "Email")
              (email-field :email))

group-check-box

(group-check-box attr-map? name value label)(group-check-box attr-map? name value label checked?)

wrapper

(wrapper attr-map? label input)(wrapper attr-map? label error hint & inputs)
Returns `label` and `inputs` as well as eventual `error` and `hint`
inside a wrapper element.

### Example

    (wrapper
     (label :email "Email")
     nil
     (hint "Double-check your email address")
     (email-field {:placeholder "joe@example.com"} :email))
